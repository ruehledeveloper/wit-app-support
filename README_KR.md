# Wit: 개인정보 보호 중심 AI 분석

**Wit**("What is this?"의 약자)는 iOS를 위한 미니멀하고 프라이버시 우선의 AI 분석 도구입니다. 계정 생성이나 데이터 수집 없이 피싱 사기 식별부터 여행지 메뉴 번역까지 주변 세상을 더 쉽게 이해하도록 도와줍니다.

## 개인정보 처리방침
**최종 업데이트: 2026년 4월**

Wit은 "프라이버시 중심 설계(Privacy by Design)" 철학을 바탕으로 개발되었습니다.

- **계정 불필요:** 이메일, 전화번호, 이름 등 어떠한 개인 정보도 요구하거나 수집하지 않습니다.
- **추적 없음:** 제3자 추적이나 광고 SDK를 전혀 사용하지 않습니다.
- **데이터 서버 저장 없음:** 사용자의 개인 데이터를 당사 서버에 저장하거나 보유하지 않습니다.
- **로컬 기록:** 분석 기록은 사용자의 기기에만 저장됩니다. 앱을 삭제하면 기록도 함께 삭제됩니다.
- **보안 처리:** 이미지는 AI 분석을 위해 암호화된(HTTPS) 통로를 통해 서버로 전송됩니다. 이미지는 메모리 상에서만 처리되며, 분석이 완료된 후 서버에 저장되지 **않습니다**. 처리 완료 후 제출된 이미지나 분석 결과를 서버에 보관하지 않습니다.
- **처리의 법적 근거:** 이미지는 요청된 분석 제공(서비스 이행)을 위한 목적으로만 처리됩니다.

## 주요 기능

- **피싱 사기 식별:** 의심스러운 문자 메시지나 링크를 즉시 분석하여 위험한 피싱 시도를 감지합니다.
- **스마트한 여행 동반자:** 외국어 메뉴나 표지판을 촬영하여 문맥에 맞는 명확한 번역 결과를 얻을 수 있습니다.
- **안전 정보 확인:** 복잡한 자동차 대시보드 경고등이나 기호의 의미를 즉시 파악하여 안전을 확인하세요.
- **원활한 앱 통합:** 공유 확장 프로그램(Share Extension)을 사용하여 사진 앱이나 다른 앱에서 직접 이미지를 Wit으로 보내 분석할 수 있습니다.

## 지원 및 자주 묻는 질문 (FAQ)

### 공유 확장 프로그램(Share Extension)은 어떻게 사용하나요?
1. **사진** 앱 또는 이미지가 있는 앱을 엽니다.
2. **공유** 버튼을 누릅니다.
3. 앱 목록에서 **Wit**을 찾아 선택합니다. (목록에 보이지 않으면 "더 보기"를 눌러 Wit을 즐겨찾기에 추가하세요.)

### 분석 기능에 문제가 있습니다. 어떻게 해야 하나요?
AI 분석을 위해서는 처리 서버와의 보안 연결이 필요하므로, 안정적인 인터넷 연결 상태인지 확인해 주세요.
