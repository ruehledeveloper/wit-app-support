# Wit: Análisis de IA Privado

**Wit** (abreviatura de "¿Qué es esto?") es una herramienta de análisis de IA minimalista y centrada en la privacidad para iOS. Te ayuda a entender el mundo que te rodea —desde identificar estafas de phishing hasta traducir menús de viaje— sin necesidad de crear una cuenta ni recopilar tus datos.

## Política de Privacidad
**Última actualización: abril de 2026**

Wit se creó bajo una filosofía de "Privacidad por diseño" (Privacy by Design).

- **Sin Cuentas:** No requerimos (ni queremos) tu correo electrónico, número de teléfono ni nombre.
- **Sin Rastreo:** No utilizamos SDK de publicidad o rastreo de terceros.
- **Sin Almacenamiento de Datos:** No almacenamos ni retenemos tus datos personales en nuestros servidores.
- **Historial Local:** Tu historial de análisis se almacena exclusivamente en tu dispositivo físico. Si eliminas la aplicación, el historial desaparece.
- **Procesamiento Seguro:** Las imágenes se envían mediante una conexión cifrada (HTTPS) a nuestro servidor para el análisis de IA. Las imágenes se procesan en la memoria volátil y **no** se almacenan en nuestros servidores una vez finalizado el análisis. No conservamos las imágenes enviadas ni los resultados del análisis tras completar el proceso.
- **Base Legal para el Procesamiento:** Las imágenes se procesan únicamente para proporcionar el análisis solicitado (ejecución de un servicio).

## Funciones Clave

- **Identificar Estafas:** Analiza instantáneamente mensajes de texto o enlaces sospechosos para detectar intentos de phishing de alto riesgo.
- **Compañero de Viaje Inteligente:** Toma una foto de un menú o letrero extranjero para obtener una traducción clara y consciente del contexto.
- **Seguridad de un Vistazo:** Comprende las alertas complejas del tablero del automóvil o las luces de advertencia al instante.
- **Integración Perfecta:** Usa la Extensión de Compartir de Wit para analizar imágenes directamente desde tu aplicación Fotos o cualquier otra aplicación en tu iPhone.

Aceptas no utilizar Wit para fines ilegales, dañinos o abusivos. Esto incluye, entre otros, el análisis o la distribución de contenido que sea ilegal, infrinja los derechos de terceros o viole las leyes y regulaciones aplicables.

Wit se reserva el derecho de tomar medidas técnicas razonables para proteger el servicio contra el mal uso, incluyendo la limitación o restricción del acceso cuando sea factible.

### Política de Uso Justo
Wit está diseñado para el uso individual y personal de los usuarios. Aunque ofrecemos niveles de análisis "ilimitados", estos están sujetos a nuestra Política de Uso Justo para garantizar la estabilidad del servicio y evitar el abuso.

Las suscripciones están destinadas a un único usuario humano, no para el procesamiento automatizado, robótico o masivo de imágenes. Wit se reserva el derecho de limitar o restringir temporalmente el acceso si los patrones de uso sugieren actividad no humana o abuso del sistema.

Esta política garantiza que podamos seguir ofreciendo análisis de IA de alta calidad a toda la comunidad a un precio justo.
