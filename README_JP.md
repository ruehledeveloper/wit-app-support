# Wit：プライベートAI分析

**Wit**（「What is this?」の略）は、iOS向けのミニマルでプライバシーを最優先したAI分析ツールです。アカウント登録やデータ収集を一切行うことなく、身の回りのことを理解するのに役立ちます。

---

## プライバシーポリシー
**最終更新日：2026年4月**

- **アカウント不要:** 登録や収集はいたしません。
- **トラッキングなし:** 第三者のトラッキングや広告SDKは使用していません。
- **安全な処理:** 画像は暗号化（HTTPS）された状態で送信されます。分析完了後にサーバーに保存されることは**ありません**。

## 利用規約とフェアユース・ポリシー (FUP)

### 許諾事項
違法、有害、または乱用目的でWitを使用しないことに同意するものとします。これには、違法なコンテンツ、他者の権利を侵害するコンテンツ、または適用される法令に違反するコンテンツの分析や配布が含まれますが、これらに限定されません。

Witは、サービスの悪用から保護するために、可能な範囲でアクセスを制限または禁止することを含め、合理的な技術的措置を講じる権利を留保します。

### フェアユース・ポリシー
Witは、一般ユーザーによる個人的な利用を目的として設計されています。「無制限」の分析プランを提供していますが、これらはサービスの安定性を維持し、悪用を防止するためにフェアユース・ポリシーの対象となります。

サブスクリプションは一人の人間による利用を想定しており、自動化された処理、ロボットによる処理、または大量の画像処理を目的としたものではありません。利用パターンが非人間的な活動やシステムの悪用を示唆する場合、Witは一時的にアクセスを制限または禁止する権利を留保します。

このポリシーにより、高品質なAI分析をコミュニティ全体に適正な価格で提供し続けることが可能になります。
