# Wit：プライベートAI分析

**Wit**（「What is this?」の略）は、iOS向けのミニマルでプライバシーを最優先したAI分析ツールです。アカウント登録やデータ収集を一切行うことなく、フィッシング詐欺の特定から旅行先でのメニュー翻訳まで、身の回りのことを理解するのに役立ちます。

[App Storeでダウンロード](#) *(承認後にリンクが有効になります)*

---

## プライバシーポリシー
**最終更新日：2026年4月**

Witは「プライバシー・バイ・デザイン」の理念に基づいて構築されています。

- **アカウント不要:** メールアドレス、電話番号、名前などの登録は不要です（収集もいたしません）。
- **トラッキングなし:** 第三者のトラッキングや広告SDKは使用していません。
- **データ保存なし:** サーバー上に個人データを保存または保持することはありません。
- **ローカル履歴:** 分析履歴は、お客様のデバイス内にのみ保存されます。アプリを削除すると、履歴も削除されます。
- **安全な処理:** 画像は暗号化（HTTPS）された状態でサーバーに送信されます。メモリ内で処理され、分析完了後にサーバーに保存されることは**ありません**。
- **処理の法的根拠:** 画像は、要求された分析を提供するためのみに処理されます（サービスの履行）。

---

## 主な機能

- **詐欺の特定:** 不審なテキストメッセージやリンクを即座に分析し、高リスクのフィッシング詐欺を検知します。
- **スマートな旅行ガイド:** 外国のメニューや看板の写真を撮るだけで、文脈を考慮した明確な翻訳を表示します。
- **一目でわかる安全確認:** 車のダッシュボードのアラートや警告灯の意味を即座に理解できます。
- **シームレスな連携:** Wit共有エクステンションを使用して、写真アプリやその他のiPhoneアプリから直接画像を分析できます。

## サポートとFAQ

### 共有エクステンションの使用方法
1. **写真**アプリ、または画像のあるアプリを開きます。
2. **共有**ボタンをタップします。
3. アプリの一覧から**Wit**を探してタップします（表示されない場合は「その他」をタップしてWitをお気に入りに追加してください）。

### 分析がうまくいかない場合は？
AI分析にはサーバーとの安全な接続が必要なため、安定したインターネット接続があることを確認してください。
